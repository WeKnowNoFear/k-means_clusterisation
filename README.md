# K-Means Clustering (OpenMP)

Проект реализует алгоритм кластеризации **K-means** с использованием параллелизации через **OpenMP**, а также автоматический подбор оптимального числа кластеров и вычисление метрик качества.

## 📌 Возможности

- Реализация алгоритма **K-means**
- Инициализация центроидов методом **k-means++**
- Параллельная обработка с использованием **OpenMP**
- Подбор оптимального числа кластеров (`k`)
- Расчёт метрик:
  - **WCSS (Within-Cluster Sum of Squares)**
  - **Silhouette score**
- Сохранение результатов в CSV-файлы

## 🧠 Описание алгоритмов

### K-means

1. Инициализация центроидов (k-means++)
2. Назначение точек к ближайшему центроиду
3. Пересчёт центроидов
4. Повторение до сходимости или max итераций

---

### K-means++

Метод улучшенной инициализации:
- первый центроид выбирается случайно
- остальные выбираются с вероятностью, пропорциональной расстоянию до уже выбранных центроидов

---

### Метрики

- WCSS — измеряет компактность кластеров  
- Silhouette score — оценивает качество кластеризации:
  - ближе к 1 → хорошо
  - около 0 → кластеры перекрываются
  - отрицательное → плохое разбиение
